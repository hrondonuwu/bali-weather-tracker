# Weather Tracker for Bali, Indonesia
This project is a Python-based weather tracker designed to assist in planning trips to Bali, Indonesia. By analyzing past weather trends and current forecasts, it provides the data necessary to make informed decisions about when to visit and what gear to pack.

# How it works:
- Data Sourcing: The script fetches historical daily high/low temperatures and a 7-day forecast using the Open-Meteo API.

- Analysis: It aggregates five years of historical data alongside a live 7-day forecast.

- Output: The script generates and saves the analyzed data into historical_weather.csv and forecast_weather.csv for easy inspection and portability.
